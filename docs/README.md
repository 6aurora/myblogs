# 七、运算符

## 1.1分类

* 算术运算符:  通过算术运算符组织的数据, 称之为算术表达式.比如:   int a=1; int b=2; int sum = **a+b**

* 比较运算符:  通过比较运算符组织的数据, 称之为条件表达式,  比如:   int a=1; int b=2;   boolean flag = **a>b**

* 逻辑运算符: 通过逻辑运算符组织的数据, 称之为关系表达式,  比如:   int a=1; int b=2; int c = 3  boolean flag =**(  a>b) && (c>b)**

## 1.2逻辑与和短路与（&，&&）

逻辑与（&）：如果多个条件表达式都为true，则结果为true，每个条件表达式都进行判断

短路与（&&）：如有一个条件表达式为false，则结果为false，后面的条件表达式都不在进行判断

## 1.3逻辑或和短路或（|，||）

逻辑或（|）：如果有一个条件表达式为true，则结果为true，每个条件表达式都进行判断

短路或（||）：如有一个条件表达式为true，则结果为true，后面的条件表达式都不在进行判断

# 十一、数组的扩容(理解)

扩容: 改变数组的长度

分析: 数组的长度是固定, 该如何扩容.

思路是这样的:

1. 新建一个数组,指定扩容后的长度
2. 将原有数组的数据设置到新的数组里面
3. 将新建的数组再赋值给原有数组,完成原有数组的扩容



# 十二、数组的添加和删除操作(理解)

## 1.1指定位置添加

````java
 //指定位置添加
    public static int[] add(int[] oldArr,int index,int data){
        //创建一个新的数组，数组长度比原数组多一
        int[] newArr = new int[oldArr.length+1];
        //遍历旧数组，将旧数组的值插入新数组
        for (int i = 0; i < oldArr.length; i++) {
            //判断索引是否是插入位置
            if (i < index){
                //插入位置之前，数组索引不变
                newArr[i] = oldArr[i];
            }else {
                //插入位置之后，索引向后移动一位
                newArr[i+1] = oldArr[i];
            }
        }
        //插入指定的值
        newArr[index] = data;
        return newArr;
    }

````



## 1.2指定位置删除

````java
 //指定位置删除
    public static int[] delete(int[] oldArr,int index){
        //创建一个新的数组，数组长度比原数组少一
        int[] newArr = new int[oldArr.length-1];
        //遍历旧数组，将旧数组的值插入新数组
        for (int i = 0; i < newArr.length; i++) {
            //判断索引是否是删除位置
            if (i < index){
                //删除位置之前，数组索引不变
                newArr[i] = oldArr[i];
            }else {
                //删除位置之后，索引向前移动一位
                newArr[i] = oldArr[i+1];
            }
        }
        return newArr;
    }
````

# 十六、排序算法

## 1.1冒泡排序

```java
/**
 * 冒泡排序
 * 两两相比，大的往后交换，每轮选出最大的，下一轮比较的数少一
 * 向后冒泡
 */
public class Demo1 {
    public static void main(String[] args) {
        //定义一个数组
        int[] arr = {1,4,7,2,8,5};
        for (int i =0; i<arr.length-1;i++){     //比较的轮次（每轮选出一个最大的，下一轮剩下的比较）
            for (int j=0;j<arr.length-1-i;j++){     //每轮两两比较的次数
                if (arr[j] > arr[j+1]){
                    //交换位置
                    int temp = arr[j];
                    arr[j] = arr[j+1];
                    arr[j+1] = temp;
                }
            }
        }
        System.out.println(Arrays.toString(arr));
    }
}

```

## 1.2选择排序

````java
/**
 * 选择排序，每轮选出一个最小的数与第一个交换位置，下一轮比较的数少一
 */
public class Demo2 {
    public static void main(String[] args) {
        //定义一个数组
        int[] arr = {1,4,7,2,8,5};
        for (int i=0;i<arr.length-1;i++){       //循环的次数（每轮选出一个最小的放在第一位，剩下的下一轮）
            for (int j=i+1;j<arr.length;j++){     //与第一位的值比较，交换位置
                if (arr[i] > arr[j]){
                    int temp = arr[i];
                    arr[i] = arr[j];
                    arr[j] = temp;
                }
            }
        }
        System.out.println(Arrays.toString(arr));
    }
}
````

## 1.3插入排序

````java
/**
 * 插入排序(保证前面的数都是正确的顺序，每次一个数与前面的数比较，插入正确的位置)
 */
public class Demo3 {
    public static void main(String[] args) {
        //定义一个数组
        int[] arr = {1,4,7,2,5,8};
        for (int i = 1;i <arr.length;i++){      //总的比较轮次（几个数就比较几轮）
            /**
             * j = 1表示左边只有一个数，后面数只要与前面比较一次
             */
            for (int j=i;j>0;j--){      //后面的数与前面数比较，放入正确的位置
                if (arr[j] < arr[j-1]){
                    //交换位置
                    int temp = arr[j];
                    arr[j] = arr[j-1];
                    arr[j-1] = temp;
                }
            }
        }
        System.out.println(Arrays.toString(arr));
    }
}
````

成员变量不允许重名

局部变量不允许重名,如果在不同的方法内部,可以定义重名的局部变量.

成员变量和局部变量可以重名,在使用时,就近原则.

## 1.3默认值

成员变量都有默认值.

如果成员变量是整数类型:  默认值是0

如果成员变量是浮点数类型:  默认值是0.0

如果成员变量是布尔类型:  默认值是false

如果成员变量是字符类型:  默认值是空字符

如果成员变量是String ,默认值是null

如果成员变量是引用类型,如果没有赋值,默认值都是null

局部变量没有默认值,所以在使用局部变量时,必须先赋值

## 1.4存储位置

局部变量存储在栈中，成员变量存储在堆中

# 四、构造方法(掌握)

## 1.1特点

创建对象时调用构造方法

````java
类名 对象 = new 类名（）
````

## 1.2类别

* 无参构造方法
* 有参构造方法

## 1.3注意细节

没有有参构造器时，无参构造器默认存在，有了有参构造器后，默认的无参构造器就被替代了
